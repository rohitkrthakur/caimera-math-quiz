# Competitive Math Quiz

A real-time multiplayer math quiz application built as a full stack assignment.

Users can join the same game room, solve math questions, and compete to answer first. The first correct answer wins the round and the next question is generated automatically.

---

## Tech Stack

Frontend
- Next.js 14
- TypeScript
- Tailwind CSS
- Socket.io Client

Backend
- Node.js
- Express.js
- Socket.io

---

## Features

- Real-time multiplayer gameplay
- Dynamic math question generation
- First correct answer wins
- Live leaderboard / high scores
- Automatic next question generation
- Socket-based communication
